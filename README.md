# Personal Football Referee Manage System Backend

[![Build Status](https://travis-ci.org/SoraYama/Football-Server.svg?branch=master)](https://travis-ci.org/SoraYama/Football-Server)

## QuickStart

### Development

```bash
$ npm i
$ npm run dev
```

### Deploy

```bash
$ npm start
```

### Npm Scripts

- Use `npm test` to run unit test

### Requirement

- Node.js 8.x
- Typescript 2.8+
